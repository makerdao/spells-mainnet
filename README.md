# dss-launch

Staging repo for MakerDAO weekly executive spells.

### Getting Started

```
$ git clone git@github.com:makerdao/dss-launch.git
$ dapp update
```

### Build

```
$ make build
```

### Test

Set `ETH_RPC_URL` to a Mainnet node.

```
$ ETH_RPC_URL=<Mainnet>
$ make test
```

### Deploy

```
$ make deploy
```
